# Code of Conduct

## Our Standard

Be respectful, constructive, and technically honest.

This project is small and pragmatic. Harassment, discrimination, sealioning, personal attacks, and other time-wasting nonsense are not welcome.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that make the project worse instead of better.

Security reports should follow [SECURITY.md](SECURITY.md).
